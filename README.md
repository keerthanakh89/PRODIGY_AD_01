# 🧮 PRODIGY_AD_01 - Simple Calculator App

A simple Calculator Android App built using **Java**
as Task 1 of my Android Development Internship at **Prodigy InfoTech**.

## 📱 Features
- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division (with divide by zero handling)
- ⚠️ Empty input validation with Toast messages
- 🎨 Clean and simple UI

## 🛠️ Tech Stack
- **Language:** Java
- **IDE:** Android Studio
- **UI:** XML Layouts
- **Components:** EditText, Button, TextView

4. Run on emulator or physical device
